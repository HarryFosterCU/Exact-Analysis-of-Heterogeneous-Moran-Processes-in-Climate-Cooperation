\section{Model}

Let us assume

\begin{itemize}
    \item \(N\) \textbf{ordered} individuals
    \item \(k\) types \(A_1, A_2, \dots A_k\)
    \item State \(v\in S = [1,\dots,k] ^ N\) so that individual \(i\) is of type \(v_i\). 
          Note that \(|S|=k^N\)
    \item Fitness function \(f:S\to \mathbb{R}^N\)
\end{itemize}
